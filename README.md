# Biblioteca API

Mini projeto de uma API para gerenciar uma biblioteca de livros. O servidor foi feito em Flask e o client em Python usando a biblioteca requests.



## Instalacao

Instale as dependencias:

```
pip install -r requirements.txt
```

## Como rodar

Em um terminal, suba o servidor:

```
python server.py
```

O servidor sobe em `http://127.0.0.1:5000`.

Em outro terminal, rode o client:

```
python client.py
```

O client abre um menu interativo para listar, buscar, cadastrar, atualizar e remover livros.
